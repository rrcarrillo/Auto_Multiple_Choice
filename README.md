# Auto Multiple Choice
Manuales del programa Auto Multiple Choice (AMC)

En este repositorio se puede encontrar:
- Un manual para instalar el programa AMC en Windows usando el Subsistema de Windows para Linux (WSL).
- Un manual para usar AMC para escribir y coregir automáticamente exámenes tipo test.
